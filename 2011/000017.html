Hello Anxo,

great that you like Pondus that you'd like to translate it, thanks a
lot for the offer!

You can always get the current version of the po files at
https://bitbucket.org/eike/pondus/src/tip/po/

As you can see there, a Spanish (Castilian) translation exists already,
but not a Galician one. Maybe you can use the Castilian one as a basis?
Then, you can send the po file to the mailing list.

Alternatively, you can also translate the current version online at
https://www.transifex.net/projects/p/pondus/resource/translations/

From there, I can automatically integrate the translations into the
source repository.

If you have further questions, don't hesitate to ask.

Thanks again and all the best from Germany,

Eike
