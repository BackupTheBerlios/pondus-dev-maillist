Hi all,

I just tagged the release candidate for pondus 0.8.0 containing the
following new features:

 * Add options to track bodyfat, muscle and water percentage (can be
   activated in preferences)
 * Add option to add notes to datasets (can be activated in preferences)
 * Plot bodyfat, muscle or water using second y-axis in graph
 * Set default plot range to "Last Year'' and replace "Last 3 months''
   with "Last 6 months''
 * Display bodyfat, muscle, water and notes as tooltips in main window

I plan to release the final version in a week or two, so now would be
the perfect time to update translations, test the new features or write
comments ;-)


1) translation updates

a) Just go to
   https://www.transifex.net/projects/p/pondus/resource/translations/
   and update the translation

or

b) Get the .po file from https://bitbucket.org/eike/pondus/src/tip/po/
   and send the updated file to me.


2) testing for bugs

Get the release candidate at https://bitbucket.org/eike/pondus/downloads
unpack the source and execute pondus.py. There is no need to permanently
install the release candidate.


Thanks a lot,

Eike
